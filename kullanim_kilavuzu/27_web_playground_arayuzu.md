# Web Playground ve Tarayıcı Motoru

TİLK kodlarını tarayıcıda denemek için playground arayüzü sunulur.

## Özellikler
- **Saf JS Motoru**: Sunucuya istek atmadan anında tarayıcıda çalışır.
- **AST Görselleştirici**: Kodun soyut sözdizimi ağacını gösterir.
- **Token Tablosu**: Lexer analizini şemalaştırır.
